## 🎯 ゲームシステム

### 敵の種類（10種類）

1. **🔴 基本敵（basicEnemy）**
   - 直線的に降下
   - 基本スコア: 10点

2. **� 追跡敵（followEnemy）**
   - プレイヤーを追跡
   - スコア: 15点

3. **� 爆発敵（explodeEnemy）**
   - 撃破時に爆発エフェクト
   - スコア: 20点

4. **� ジグザグ敵（zigzagEnemy）**
   - 左右にジグザグ移動
   - スコア: 25点

5. **🟣 高速敵（fastEnemy）**
   - 高速で降下
   - スコア: 30点

6. **🟢 ライム敵（limeEnemy）**
   - 4方向に弾を発射
   - スコア: 35点

7. **🛡️ シールド敵（shieldEnemy）**
   - 2発で破壊される耐久力
   - スコア: 40点

8. **💥 分裂敵（splittingEnemy）**
   - 撃破時に2つの小さな敵に分裂
   - スコア: 45点（本体） + 15点（分裂体）

9. **⚡ テレポート敵（teleportEnemy）**
   - ランダムにテレポート
   - スコア: 50点

10. **👑 ボス（bossEnemy）**
    - 10発で撃破
    - スコア: 100点

### パワーアップアイテム（6種類）

1. **🔫 連射アップ（rapidFire）**
   - 発射間隔を短縮
   - 効果時間: 10秒

2. **🛡️ シールド（shield）**
   - 一度だけダメージを無効化
   - 効果: 次回ダメージまで

3. **❤️ ライフアップ（extraLife）**
   - 残機を1つ追加
   - 効果: 即座

4. **⚡ レーザービーム（laserBeam）**
   - 画面を貫通する強力なレーザー
   - 効果時間: 8秒

5. **⏰ 時間減速（timeSlow）**
   - 敵の動きを半分に減速
   - 効果時間: 10秒

6. **💰 2倍スコア（doubleScore）**
   - 獲得スコアが2倍になる
   - 効果時間: 12秒
